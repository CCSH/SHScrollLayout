# SHScrollLayout
仿半糖、闲鱼、QQ联系人、豆瓣小组、微博等界面布局 一个文件实现
> 使用中如果有其他问题可以[`告诉我`](https://github.com/CCSH/SHScrollLayout/issues/new)



> 子控制器的scrollview的bounces最好不要关闭 否则会多出很多状态进行处理，我懒就不处理了
# pod导入
```
pod 'SHScrollLayout'
```
# 内部使用的组件请 pod 引用最新版本
```
pod 'SHLabelPageView'
pod 'SHScrollView'
```
